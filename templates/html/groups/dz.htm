Динамическое зондирование

Предприятие Геотест наряду с оборудованием для статического зондирования, выпускает оборудование для динамического зондирования грунтов. Это Адаптер динамического зондирования АД3 2, предназначенный для работы на буровых установках и ручной комплект динамического зондирования РДК.

Адаптер АДЗ-2

Адаптер АДЗ-2 предназначен для динамического зондирования грунтов (согласно ГОСТ 19912–2012) посредством буровых установок, которые выполняют функции опорно-анкерного устройства и привода ударного механизма адаптера. По классификации ГОСТ 19912–2012, п. 6.2.3. Таблица 3, оборудование относится к среднему (А=1120 Н/см) и тяжелому (А=2800 Н/см) типам и может поставляться в любой конфигурации.

По результатам испытаний динамическим зондированием (согласно СП 47.13330.2012 «СНиП 11–02–96 Инженерные изыскания для строительства. Основные положения» Приложение И) определяют плотность сложения, модуль деформации, угол внутреннего трения и динамическую устойчивость песчаных грунтов.

Адаптер АДЗ-2 транспортируется в разобранном виде на буровой установке и для работы монтируется/демонтируется на буровой в течение 20 минут. Передвижение по площадке буровой установки осуществляется без демонтажа оборудования. При работе, оператор посредством вращателя перемещает молот вверх. Освобождение молота для удара по наковальне происходит автоматически на заданной высоте и далее процесс повторяется. Штанги проходят сквозь центратор, установленный на буровом столе, который обеспечивает их вертикальное положение.

Устройство для измерения глубины погружения зонда выполнено из рулетки и крепится на центраторе, а измерительная лента устройства соединена с наковальней. Испытания грунтов динамическим зондированием выполняют по методике ГОСТ 19912–2012, результаты записывают в полевой журнал после каждого залога.

Для обработки результатов используется программа Geopush (входит в состав комплекта).

Комплект РДК

Комплект РДК относится к лёгкому типу (А=280 Н/см) оборудования, по классификации Таблицы 3 ГОСТ 19912–2012. Такое оборудование рекомендуется при работе в стеснённых условиях, в грунтах с Pd менее 0.7 МПа, но только после сопоставления результатов со стандартными установками.

Оба комплекта предполагают регистрацию результатов испытаний в полевом журнале. При этом исполнитель должен визуально фиксировать отсчёты, находясь при этом, в опасной близости от ударного устройства и записывать их в журнал.

Однако, результаты испытаний, записанные в журнале вручную, могут содержать ошибки и вряд ли будут убедительными для проверяющих, т.к. написать в журнале можно всё, что угодно и сколько угодно.

Не зря новый ГОСТ 19912–2012, рекомендует для электрических зондов только цифровую фиксацию результатов измерений, а Изменения 1, принятые к этому ГОСТ в 2019 году, кроме прочего, рекомендуют фиксировать показания с шагом не более 5 см. Тот же подход, на наш взгляд, должен быть и для динамических испытаний. Кроме прочего, при использовании цифровой фиксации результатов можно получить более точную информацию о грунтах, т.к в этом случае, возможна регистрация показаний после каждого удара молота, или залога с суммарным перемещением не более 5 см.

Для реализации этого подхода был разработан Комплект AdzFo на основе лазерной рулетки BOSCH GLM 50C и СПО для Андройд-устройств.

Комплект работает следующим образом. Лазерная рулетка закрепляется на центраторе комплекта АДЗ 2. В качестве мишени используется наковальня с небольшим фартуком, установленная на штанге, а для регистрации показаний (перемещений зонда) используется Андройд-устройство (планшет или телефон). При этом, лазерная рулетка соединена с телефоном по бескабельному каналу Блютуз.

После удара молота по наковальне, в момент его подъёма, Оператор по нажатию «кнопки» на экране телефона, сохраняет текущее перемещение штанги после каждого удара.

В результате, в памяти телефона формируется файл данных, который включает отметки времени и перемещения за удар. Файл данных в любой момент времени, посредством электронной почты может быть передан в программу обработки данных динамического зондирования GeoPush, установленную на ПК, для сохранения и формирования Протоколов испытаний.

При этом, исключаются всякие ошибки и приписки, т.к формировать данные «вручную» на телефоне практически невозможно.

Комплект регистрации данных динамических испытаний Комплект AdzFo может поставляться отдельно и монтироваться на различных установках динамического зондирования, в том числе и на установках ударно-вибрационного зондирования. В этом случае измеряется скорость погружения зонда, а отсчёты фиксируются автоматически через заданный интервал времени.
