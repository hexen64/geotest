Программа Pneumator предназначена для автоматизации проведения прессиометрических и штамповых испытаний с использованием комплекта КАП2, производимого АО «Геотест», а так же для дальнейшего расчета модуля деформации грунтов E, МПа по результатам проведенных испытаний.

В программе используется методика определения модуля деформации грунта согласно ГОСТ 20276.2–2020 «Грунты. Метод испытания радиальным прессиометром».

Основные возможности программы

При обработке результатов одного опыта программа позволяет:

  1. автоматическое проведение прессиометрических и штамповых испытаний используя КАП2;
  2. передача данных прессиометрического опыта в программу из контроллера ТЕСТ-ПРК;
  3. передача данных штампового опыта в программу из контроллера ТЕСТ-ШВК;
  4. коректировка начальных установок опыта;
  5. просмотр параметров комплектов и полученных данных;
  6. удобная работа с базой данных программы (хранение, сохранение и восстановление базы);
  7. построение графика прессиометрического опыта — «давление на грунт — диаметр скважины»;
  8. построение графика штампового опыта — «осадка — давление»;
  9. построение протокола штампового опыта;
  10. построение графика развития деформаций во времени;
  11. выбор масштаба графика для отчетов (ГОСТ или автоматический);
  12. расчет модуля деформации грунтов по результатам испытаний;
  13. создание отчетов для передачи проектировщикам.

Программа предусматривает вывод документов в различных форматах: PDF, MS Excel, ODF (Open Document Format), Windows Metafile, Jpeg, rich text format, текстовый документ.

Сопровождение программы

C программой вы получаете бесплатное техническое сопровождение, в которое входят:

  • получение новых версий (версии высылаются по запросу на электронную почту);
  • получение методических руководств по работе с программой.
  • консультации по эксплуатации программы (телефонные, по электронной почте и ICQ);
  • восстановление базы данных заказчика, поврежденных в результате аппаратных сбоев.
