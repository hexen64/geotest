Программа PressPW.MA предназначена для обработки данных результатов испытаний грунтов радиальным прессиометром ПЭВ-89МК.

В программе используется методика определения модуля деформации грунта согласно ГОСТ 20276.2–2020 «Грунты. Метод испытания радиальным прессиометром».

Основные возможности программы

При обработке результатов одного опыта программа позволяет:

  1. автоматическая загрузка данных прессиометрических испытаний из контроллера ТЕСТ-ПРК;
  2. возможность коректировать начальные установки опыта;
  3. возможность просмотра полной информации о прохождении опыта и параметрах используемого прессиометра;
  4. удобная работа с базой данных программы (хранение, сохранение и восстановление базы);
  5. построение графика прессиометрического опыта — «давление на грунт — диаметр скважины»;
  6. расчет модуля деформации грунтов по результатам испытаний;
  7. создание отчетов для передачи проектировщикам.

Программа предусматривает вывод документов в различных форматах: MS Excel, Windows Metafile, Jpeg, rich text format, текстовый документ, CSV, HTML.

Сопровождение программы

C программой вы получаете бесплатное техническое сопровождение, в которое входят:

  • получение новых версий (версии высылаются по запросу на электронную почту);
  • получение методических руководств по работе с программой.
  • консультации по эксплуатации программы (телефонные, по электронной почте и ICQ);
  • восстановление базы данных заказчика, поврежденных в результате аппаратных сбоев.
