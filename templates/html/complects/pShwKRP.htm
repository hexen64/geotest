Программа Штамп-КРП предназначена для автоматической обработки результатов штамповых опытов проведенных штампом ШВ60 с комплектом регистрации КРП1, разработанных в АО «Геотест».

В программе используется методика определения модуля деформации грунта согласно ГОСТ 20276–2012 «Грунты. Методы полевого определения характеристик деформируемости».

Основные возможности программы

  1. автоматическая передача данных штампового опыта в программу из контроллера ТЕСТ-КРП;
  2. возможность коректировать начальные установки опыта;
  3. возможность просматривать параметры комплекта и полученные данные;
  4. удобная работа с базой данных программы (хранение, сохранение и восстановление базы);
  5. построение графика штампового опыта — «осадка — давление»;
  6. возможность выбора масштаба графика для отчетов (ГОСТ или автоматический);
  7. расчет модуля деформации грунтов по результатам испытаний;
  8. создание отчетов для передачи проектировщикам.

Программа предусматривает вывод документов в различных форматах: MS Excel, Windows Metafile, Jpeg, rich text format, текстовый документ.

Примеры некоторых отчетов (в формате PDF)

  • Паспорт штампового опыта;
  • Протокол штампового опыта;
  • Развитие деформаций во времени;

Сопровождение программы

C программой вы получаете бесплатное техническое сопровождение, в которое входят:

  • получение новых версий (версии высылаются по запросу на электронную почту);
  • получение методических руководств по работе с программой.
  • консультации по эксплуатации программы (телефонные, по электронной почте и ICQ);
  • восстановление базы данных заказчика, поврежденных в результате аппаратных сбоев.
