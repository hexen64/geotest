Текущая версия: v2.6

Программа Impeller 2.0 предназначена для обработки результатов испытаний слабых грунтов методом вращательного среза cдвигомером-крыльчаткой, выполненного в соответствии с ГОСТ 20276.5–2020 «Грунты. Метод вращательного среза» и методом статического зондирования, выполняемым ручным пенетрометромв соответствии с ГОСТ 19912–2012 «Грунты. Методы полевых испытаний статическим и динамическим зондированием».

Программа позволяет обработать данные испытаний и получить отчетные документы для передачи проектировщикам. По результатам испытаний крыльчаткой выполняется расчет сопротивления грунта срезу и формируется отчет в соответствии с ГОСТ 20276.5–2020. Для пенетрометра выполняется расчет модуля деформации, угла внутреннего трения, удельного сцепления в соответствии с СП 446.1325800.2019 «Инженерно-геологические изыскания для строительства. Общие правила производства работ». Приложение Ж и формируется отчет по ГОСТ 19912–2012.

Программа основана на ГОСТ 20276.5–2020 «Грунты. Метод вращательного среза», ГОСТ 19912–2012 «Грунты. Методы полевых испытаний статическим и динамическим зондированием» и СП 446.1325800.2019 «Инженерно-геологические изыскания для строительства. Общие правила производства работ».

Основные возможности программы

Крыльчатка

При обработке результатов одного опыта программа позволяет:

  1. Cформировать таблицу «Журнал испытания грунта методом вращательного среза» по ГОСТ 20276.5–2020, Приложение Б.
  2. Построить эпюру изменения сопротивления сдвигу по глубине.

для объекта:

  1. Сформировать сводную таблицу «Журнал испытания грунта методом вращательного среза» (по ГОСТ 20276.5–2020, Приложение Б) по всем выбраным опытам.
  2. Рассчитать нормативные и расчетные значения сопротивления срезу, коэффициент вариации.
  3. Построить эпюру изменения расчетного и норматичного сопротивления сдвигу по глубине.

Пенетрометр

При обработке результатов одного опыта программа позволяет:

  1. Cформировать таблицу «Журнал испытания грунта методом статического зондирования» по ГОСТ 19912–2012, Приложение Б;
  2. Построить график изменения сопротивления конуса по глубине погружения пенетрометра по ГОСТ 19912–2012, Приложение Г;
  3. Рассчитать значения модуля деформации, угла внутреннего трения и удельного сцепления грунта.

Программа предусматривает вывод документов в различных форматах: MS Excel, MS Word (табличный) PDF, JPG, BMP, RTF, CSV, HTML, текстовый документ.

Вы можете посмотреть снимки рабочих окон программы: главное окно программы, проведение испытания и работа с графиком прессиометрического опыта.
