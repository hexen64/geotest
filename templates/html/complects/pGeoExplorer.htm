Программы серии Geoexplorer предназначены для обработки результатов статического зондирования грунтов и используются с комплектами аппаратуры ТЕСТ-К2М, ТЕСТ-12, ТЕСТ-К4М и ТЕСТ К4 CPTU, разработанными АО «Геотест».

Программа позволяет обработать данные статического зондирования и получить отчетные документы для передачи проектировщикам.

Основу программы составляют следующие нормативные документы:

  • СП 11–105–97 Часть I (Приложение И);
  • СП 446.1325800.2019 «Инженерные изыскания для строительства. Основные положения»;
  • ГОСТ 20522–2012 «Методы статистической обработки результатов испытаний»;
  • СП 24.13330.2011 «СНиП 2.02.03–85 Свайные фундаменты»;
  • СП 50–102–2003 «Проектирование и устройство свайных фундаментов»;
  • МГСН 2.07–01 «Московские городские строительные нормы. Основания, фундаменты и подземные сооружения».

Публикации

GEOEXPLORER для обработки результатов статического зондирования. Инженерные изыскания. Июнь 2008.

Основные возможности программы

При обработке результатов одного опыта программа позволяет:

  • построить графики qc, МПа и fs, кПа;
  • определить вид и состояние грунтов;
  • оценить нормативные значения прочностных характеристик С, кПа и φ, град;
  • оценить модуль деформации Е, МПа;
  • выделить ИГЭ (инженерно-геологические элементы);
  • описать ИГЭ (cправочник ИГЭ для объекта);
  • получить для ИГЭ расчетные характеристики;
  • построить геологический разрез скважины в точке зондирования;
  • рассчитать частные значения предельного сопротивления забивных и буровых висячих свай.

Для объекта:

  • построить геологический разрез площадки по результатам зондирования;
  • рассчитать нормативные и расчетные характеристики для ИГЭ;
  • рассчитать несущую способность забивных и буровых висячих свай по объекту.
  • рассчитать несущую способность винтовых и забивных круглых свай по объекту.

Программа предусматривает вывод документов в различных форматах: PDF, BMP, EMF, MS Excel, Windows Metafile, Jpeg, rich text format, текстовый документ, CSV, HTML.

Примеры некоторых отчетов (в формате PDF)

Для опыта:

  • Паспорт статического зондирования;
  • Отчет по ГОСТ 19912–2012;
  • Протокол опыта CPTU (только v.2.xx);
  • Геологическая колонка;
  • Отчет по ГОСТ 19912–2012 с геологической колонкой;
  • Геологический разрез скважины;
  • Несущая способность свай.

Для объекта:

  • Геологический разрез площадки по результатам зондирования;
  • Сводная таблица физико-механических характеристик грунтов;
  • Нормативные и расчетные характеристики ИГЭ;
  • Несущая способность свай по объекту.

Сопровождение программы

При покупке программы вы получаете бесплатное техническое сопровождение, в которое входят:

  • получение новых версий (версии высылаются по запросу на электронную почту);
  • получение методических руководств по работе с программой.
  • консультации по эксплуатации программы (телефонные и по электронной почте);
  • восстановление базы данных заказчика, поврежденных в результате аппаратных сбоев.
