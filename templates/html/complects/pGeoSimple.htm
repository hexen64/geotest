Текущая версия: v1.5

Программа GeoSimple предназначена для ввода архивных данных, сохранившихся в полевых журналах, а так же для конвертации данных статического зондирования полученных с помощью различной аппаратуры, в формат GXL для передачи в программу GeoExplorer.

GeoSimple принимает данные из широкого спектра форматов:

  • данные всех контроллеров серии «ТЕСТ» (GEO);
  • Geotechnical Exchange Format for CPT-data (GEF);
  • данные аппаратуры «ENVI» (TXT);
  • данные оборудования «ФУГРО» (A00);
  • табличные данные Microsoft Excel, Comma-Separated Values (XLS, CSV);
  • текстовые данные (TXT) с различными разделителями.

В программе GeoSimple данные могут быть отредактированы или добавлены вручную. Значения сопротивления конуса вводятся в МПа, значения сопротивления муфты в кПа. При работе с аппаратурой К4 значения порового давления вводятся в МПа, значения угла отклонения (инклинометр) вводятся в целых градусах.

После загрузки, в основном окне программы можно изменить данные в таблице отсчетов, задать номер опыта, дату опыта, исполнителя, привязку опыта, абсолютную отметку устья, максимальное усилие тарировки конуса и муфты, шаг зондирования, начальную глубину зондирования. После чего можно сгенерировать файл gxl для дальнейшей обработки данных в программе GeoExplorer.

Так же, программа предоставляет возможность получения простого (без обработки) отчета в виде графика профиля зондирования (по ГОСТ 19912–2012). Отчёт можно распечатать, сохранить в формате PDF или JPG.

Результатом работы GeoSimple является GXL файл, который может быть передан в программу в программу GeoExplorer.


Сопровождение программы
