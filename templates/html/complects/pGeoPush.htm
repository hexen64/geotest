Программы серии Geoepush предназначены для обработки результатов динамического зондирования грунтов и используются с комплектами оборудования Адаптер динамического зондирования АДЗ-2 и Ручной динамический комплект РДК, разработанными АО «Геотест».

Программа позволяет обработать данные динамического зондирования и получить отчетные документы для передачи проектировщикам.

Основу программы составляют следующие нормативные документы:

  • СП 47.13330.2012 СНиП 11–02–96 «Инженерные изыскания для строительства. Основные положения»;
  • ГОСТ 20522–2012 «Методы статистической обработки результатов испытаний»;

Основные возможности программы

При обработке результатов одного опыта программа позволяет:

  • оценить нормативные значения прочностных характеристик песчанных грунтов, φ, град;
  • оценить модуль деформации песчанных грунтов Е, МПа;
  • определить вероятность разжижения песков;
  • выделить ИГЭ (инженерно-геологические элементы);
  • описать ИГЭ (cправочник ИГЭ для объекта);
  • получить для ИГЭ расчетные характеристики;
  • построить геологический разрез скважины в точке зондирования;

Для объекта:

  • построить геологический разрез площадки по результатам зондирования;
  • рассчитать нормативные и расчетные характеристики для ИГЭ;

Программа предусматривает вывод документов в различных форматах: MS Excel, Windows Metafile, Jpeg, rich text format, текстовый документ, CSV, HTML.

Примеры некоторых отчетов (в формате PDF)

Для опыта:

  • Паспорт динамического зондирования;
  • Отчет по ГОСТ 19912–2012 v.2;
  • Отчет ГОСТ 19912–2012 + разрез;
  • Характеристики ИГЭ в точке зондирования;

Для объекта:

  • Геологический разрез по данным зондирования;
  • Справочник проб для объекта;
  • Таблица нормативных и расчетных характеристик грунтов;
  • Сводная таблица физико-механических характеристик грунтов.

Сопровождение программы

C программой вы получаете бесплатное техническое сопровождение, в которое входят:

  • получение новых версий (версии высылаются по запросу на электронную почту);
  • получение методических руководств по работе с программой.
  • консультации по эксплуатации программы (телефонные, по электронной почте и ICQ);
  • восстановление базы данных заказчика, поврежденных в результате аппаратных сбоев.
