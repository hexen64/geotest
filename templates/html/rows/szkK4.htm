Регистрирующий прибор (контроллер ТЕСТ-К4) предназначен для индикации, сохранения и передачи результатов зондирования на персональный компьютер для последующей обработки.

Контроллер имеет функции самодиагностики измерительного тракта.

Прибор защищен от недопустимой полярности питающего напряжения.

Устройство и работа

Контроллер ТЕСТ-К4 имеет четыре основных режима работы:

  • рабочий режим (запись данных зондирования в ОЗУ);
  • режим передачи данных зондирования в компьютер;
  • режим записи данных на USB-ФЛЕШ-ДИСК;
  • режим очистки памяти.

На передней панели измерительного прибора расположены: дисплей и пять кнопок управления. На верхней стенке контроллера расположены разъемы для подключения питания, зонда и радиокнопки.

После подачи питания, контроллер проводит самодиагностику, и автоматически переходит в РАБОЧИЙ РЕЖИМ.

В РАБОЧЕМ РЕЖИМЕ, после ввода параметров опыта (номер, шаг отсчетов и глубина зондирования), текущие отсчеты сохраняются в контроллере при нажатии кнопки «ПУСК». Одновременно, они распечатываются на минипринтере. Управлять контроллером для записи данных зондирования можно при помощи Радиокнопки РК-11.

При выборе оператором, режима ПЕРЕДАЧИ ДАННЫХ, данные передаются в компьютер для последующей обработки.

Режим ОЧИСТКИ ПАМЯТИ предусмотрен для удаления из контроллера обработанных данных зондирования.

К контроллеру предусмотрено подключение Формирователя Отсчетов. В этом случае, сохранение отсчетов происходит автоматически по командам формирователя отсчетов с любым шагом из ряда: 2.5, 5, 10 и 20 см.
