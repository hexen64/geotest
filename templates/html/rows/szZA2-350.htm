Предназначен для измерения усилий действующих на зонд при его погружении в исследуемый грунт. Тензометрический зонд состоит корпуса, конуса и муфты трения. Внутри муфты трения установлен тензодинамометр, который при погружении зонда в грунт деформируется как от действия грунта на конус зонда, так и от силы трения, действующей на муфту трения. Сигнал с тензодинамометра по кабелю, проходящему внутри зондировочных штанг передаётся на измерительный прибор.

Основные параметры зондов 2-го типа регламентируются ГОСТ 19912–2012, где приведены геометрические размеры зондов и требования к основной погрешности измерения показателей сопротивления грунта. Допустимые диапазоны измерения усилий по конусу и муфте трения зонда определяются их конструктивными особенностями.

Основным типом зонда, предназначенного для работы с комплектом ТЕСТ, являются зонды исполнения 1 - А2–350–50/20. Эти зонды калибруются на предприятии-изготовителе на диапазоны измерения 50 МПа по каналу «Конус» и 571 кПа по каналу «Муфта». Для работы со слабыми грунтами поставляются зонды исполнения 2 - А2–350–25/20, которые калибруются на диапазоны измерения по каналу «Конус» — 25 МПа, а по каналу «Муфта» — 571 кПа.
