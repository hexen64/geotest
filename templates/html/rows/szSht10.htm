Штанги КТ имеют коническую резьбу и проходят дополнительную термообработку (обладают повышенными механическими свойствами). На штангах выполнены пазы для извлечения колонны штанг по окончании опыта. Предназначены для погружения зонда при проведении статического зондирования грунтов. Внутри зондировочных штанг проходит кабель зонда, который обеспечивает передачу сигнала с тензометрического зонда на измерительный прибор.
